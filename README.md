## Lancement en local

Pour consulter le site localement, suivez ces étapes :

1. **Téléchargez les fichiers** : Clonez ou téléchargez ce dépôt GitHub sur votre ordinateur.

2. **Ouvrez le dossier** : Naviguez vers le dossier racine du projet (par exemple, `MartinFaurePortfolio`).

3. **Lancez un serveur local** (recommandé pour une meilleure expérience) :
   - Si vous avez Python installé : Ouvrez un terminal dans le dossier et exécutez `python -m http.server 8000` (ou `python3` si nécessaire), puis ouvrez `http://localhost:8000` dans votre navigateur.
   - Si vous utilisez VS Code : Installez l'extension "Live Server" et cliquez sur "Go Live" en bas à droite.
   - Sinon, ouvrez simplement `index.html` dans votre navigateur web (mais certaines fonctionnalités peuvent ne pas fonctionner correctement).

4. **Naviguez** : Le fichier principal est `index.html`. Vous pouvez également ouvrir d'autres pages comme `projets.html`, `blog.html`, etc.

Si vous rencontrez des problèmes, assurez-vous que tous les fichiers sont présents et que votre navigateur supporte JavaScript.
